# ASCII Rizz

Generate ASCII art from a picture.

Just upload or paste the link of the image to generate. 

## Technologies 
- HTML
- CSS
- JavaScript
- Web Assembly (Go Lang)

## Run Locally

Clone the project

```bash
  git clone https://github.com/shreverr/AsciiRizz
```

Go to the project directory

```bash
  cd AsciiRizz
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
